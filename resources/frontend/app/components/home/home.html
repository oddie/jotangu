JotAngu: Laravel + AngularJS + JWT application example

  • Backend: Laravel 5.2
  • Frontend: AngularJS
  • Implemented login and registration with email/password and social services. (Facebook, Google+, VKontakte)

You need configure your web server for splitting requests to frontend and backend. For nginx configuration is like this:

location /api { try_files $uri $uri/ /index.php?$query_string; }

location / { try_files $uri $uri/ /index.html; }
